## FIELD

```sql
FIELD(value, 		 -- значение(ия) для поиска
	  val1, valN...) -- проверяемые значения

-- возвращаются найденные значнеия
-- если не находит, вернет 0
-- поиск только по числовым или по строчным занчениям

-- ПРИМЕР
ORDER BY FIELD(id, 5, 1, 2); -- сортировка по последовательности
```
